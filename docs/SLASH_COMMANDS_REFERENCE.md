## üîç Quick Access Methods

### Method 1: List in Terminal
```bash
# Count total commands
cat .claude/settings.json | jq -r '.slash_commands[]' | wc -l

# List all commands
cat .claude/settings.json | jq -r '.slash_commands[]' | sort

# Search specific category
cat .claude/settings.json | jq -r '.slash_commands[]' | grep "sparc:"
```

### Method 2: Browse Files
```bash
# Navigate to commands directory
cd .claude/commands/

# List all command files
find . -name "*.md" | sort

# Read specific command
cat sparc/tdd.md
```

### Method 3: In Cursor
Type `/` in the chat to see autocomplete suggestions for all available slash commands.
